## **Technologies** 
- React Native
- Styled Components
- TypeScript
- React Native Vector Icons
- React Navigation
- Expo

##  Running App
- Clone Repository 
```
$ git clone https://github.com/anapaulalins/people.git

$ cd people
```

- Install the dependencies 
```
$ yarn

# or

$ npm install
```

- Run the application
```
$ expo start
```
